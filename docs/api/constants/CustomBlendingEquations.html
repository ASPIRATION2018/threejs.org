自定义混合方程常量

示例

[example:webgl_materials_blending_custom materials / blending / custom ]

使用

这些适用于所有材料类型。首先将材质的混合模式设置为 THREE.CustomBlending，然后设置所需的混合方程（Blending Equation），源因子（Source Factor）和目标因子（Destination Factor）。 var material = new THREE.MeshBasicMaterial( {color: 0x00ff00} ); material.blending = THREE.CustomBlending; material.blendEquation = THREE.AddEquation; //default material.blendSrc = THREE.SrcAlphaFactor; //default material.blendDst = THREE.OneMinusSrcAlphaFactor; //default

混合方程（Blending Equation）

THREE.AddEquation（译注：加法方程） THREE.SubtractEquation（译注：减法方程） THREE.ReverseSubtractEquation（译注：反向减法方程） THREE.MinEquation（译注：最小方程） THREE.MaxEquation（译注：最大方程）

源因子（Source Factors）

THREE.ZeroFactor（译注：对应于GL_ZERO，表示使用0.0作为因子，实际上相当于不使用这种颜色参与混合运算） THREE.OneFactor（译注：GL_ONE，表示使用1.0作为因子，实际上相当于完全的使用了这种颜色参与混合运算） THREE.SrcColorFactor（译注：GL_SRC_COLOR，表示使用源颜色的四个分量分别作为因子的四个分量） THREE.OneMinusSrcColorFactor（译注：GL_ONE_MINUS_SRC_COLOR，表示使用1.0减去源颜色的四个分量分别作为因子的四个分量） THREE.SrcAlphaFactor（译注：GL_SRC_ALPHA，表示使用源颜色的alpha值来作为因子。） THREE.OneMinusSrcAlphaFactor（译注：GL_ONE_MINUS_SRC_ALPHA，表示用1.0减去源颜色的alpha值来作为因子。） THREE.DstAlphaFactor（译注：GL_DST_ALPHA，表示使用目标颜色的alpha值来作为因子。） THREE.OneMinusDstAlphaFactor（译注：GL_ONE_MINUS_DST_ALPHA，表示用1.0减去目标颜色的alpha值来作为因子。） THREE.DstColorFactor（译注：对应于GL_DST_COLOR，表示使用目标颜色的四个分量分别作为因子的四个分量。） THREE.OneMinusDstColorFactor（译注：GL_ONE_MINUS_DST_COLOR 表示使用1.0减去目标颜色的四个分量分别作为因子的四个分量。） THREE.SrcAlphaSaturateFactor（译注：GL_SRC_ALPHA_SATURATE 表示源颜色的alpha 值和RGB值采用不同的混合因子。）

目标因子（Destination Factors）

除 THREE.SrcAlphaSaturateFactor 外，所有源因子（Source Factors）均可作为目标因子（Destination Factors）。
