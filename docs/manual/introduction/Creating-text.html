通常情况下，您可能需要在 Three.js 应用程序中使用文本——这里有几种可以这样做的方法。

1. DOM + CSS

使用 HTML 通常是添加文本的最简单和最快的方式。 这是大多数 Three.js 示例中用于（添加）描述性覆盖（式文本）的方法。

你可以添加内容到一个

<div id="info">Description</div>

同时使用 CSS 标记进行绝对定位，并且设置 z-index 属性使其位于其他元素之上， 特别是在运行 Three.js 全屏时。

#info { position: absolute; top: 10px; width: 100%; text-align: center; z-index: 100; display:block; }

2. 将文本绘制到画布上并用作为一个 [page:Texture]

如果您希望轻松地在您的 Three.js 场景中的平面上绘制文本，请使用此方法。

3. 在您最喜欢的 3D 应用程序中创建一个模型并导出到 Three.js

如果您喜欢使用 3D 应用程序并将模型导入到 Three.js，请使用此方法。

4. 程序性的文本几何图形

如果您喜欢单纯地在 Three.js 中工作，或者创建程序上的动态3D文本几何图形，你可以创建一个网格， 它的几何图形是 THREE.TextGeometry 的一个实例：

new THREE.TextGeometry( text, parameters );

然而，为了使这一工作有效， 你的 TextGeometry 需要在它的 “font” 参数上设置一个 THREE.Font 的实例。 有关如何做到这一点，对每个可接受参数的描述，以及与 Three.js 发行版本身一起提供的 JSON 字体列表的更多信息， 请参见 [page:TextGeometry] 页面。

示例

[example:webgl_geometry_text WebGL / geometry / text]
[example:canvas_geometry_text canvas / geometry / text]
[example:webgl_shadowmap WebGL / shadowmap]
