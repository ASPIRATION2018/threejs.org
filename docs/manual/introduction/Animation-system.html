[name]

概览

在three.js的动画系统中，你可以动画化模型的任何属性：[page:SkinnedMesh skinned and rigged model]的骨骼， [page:Geometry.morphTargets morph targets]，不同材质属性（颜色，透明度，布尔值），可见性和变换。动画属性可以淡入，淡出，交叉淡入淡出和扭曲。同一个对象上不同动画的权重和时间尺度可以像不同对象一样单独改变。 在同一个或不同对象上的动画可以同步。

为了在同一个动画系统（three.js动画系统[link:https://github.com/mrdoob/three.js/issues/6881 has completely changed in 2015]请注意其中过时的信息！）中实现上述效果，three.js动画系统有一个类似于Unity/Unreal 引擎4的体系。本章节简要介绍其中的主要组件以及他们是如何共同产生效果的。

动画裁剪

如果你已经成功导入和动画操作过3D对象（无所谓这个3D对象是否有骨骼或者变形目标或两者都有）- 例如，从Blender中使用[link:https://github.com/mrdoob/three.js/tree/master/utils/exporters/blender/addons/io_three Blender exporter]导出，并使用[page:JSONLoader]加载到three.js场景中 - ，加载的mesh的geometry属性之一应该有一个叫做‘animations’的数组，包含这个模型的[page:AnimationClip AnimationClips]（查看下述列表中的常见加载器）。

每个*动画裁剪*通常包含对象某个动作的数据。如果mesh是一个字符，例如，一个动画裁剪用于漫游，第二个用于跳跃，第三个用于躲闪等等。

关键帧跟踪

在这样一个*动画裁剪*内部，每个动画属性的数据都保存在一个单独的[page:KeyframeTrack]中。假设一个字符对象有一个[page:Skeleton skeleton]，一个关键帧可以保存在包含一些较低的手臂骨骼随着时间的位置变换的数据中，另一帧跟踪同一骨骼的旋转变换，第三帧跟踪其他骨骼的位置、旋转或缩放，等等。应该清楚的是，一个动画裁剪可以由许多这样的关键帧跟踪组合而成。

假设模型有[page:Geometry.morphTargets morph targets]（例如一个变形目标展示友好的表情，另一个展示愤怒的表情），每个帧包含的信息是某个变形目标的[page:Mesh.morphTargetInfluences influence]是怎样在裁剪表现间变化的。

动画混合器

数据只是构成动画的基础 - 实际的回放是通过[page:AnimationMixer]来控制的。你可以认为动画混合器不仅是一个动画的播放器，也是一个类似实际调音台硬件的模拟，他可以同时控制不同的动画，并混合和合并他们。

动画动作

*动画混合器*本身只有非常少的属性和方法，因为他可以被[page:AnimationAction AnimationActions]控制。通过确认一个*动画动作*，你可以决定某个*动画裁剪*何时在混合器中播放，暂停或停止，动画裁剪必须多久重复一次，是否应该通过渐变或时间比例来展现，以及一些其他的事情，例如交叉渐变或同步。

动画对象组

如果你想要对对象分组以便接受一个共同的动画状态，可查看这一章节 [page:AnimationObjectGroup].

支持的形式和加载器

需要注意的是，不是所有的模型格式都包含动画（OBJ即显然不支持），只有一些three.js加载器支持[page:AnimationClip AnimationClip]序列。一些确实支持这些动画类型：
  • [page:JSONLoader THREE.JSONLoader]
  • [page:ObjectLoader THREE.ObjectLoader]
  • THREE.BVHLoader
  • THREE.ColladaLoader
  • THREE.FBXLoader
  • [page:GLTFLoader THREE.GLTFLoader]
  • THREE.MMDLoader
  • THREE.SEA3DLoader
需要注意的是，3ds max和Maya当前还不能导出多个动画（即不在同一个时间线上的动画）到一个单独的文件中。

示例

var mesh; // 构造一个动画混合器，并且获得动画裁剪示例的列表 var mixer = new THREE.AnimationMixer( mesh ); var clips = mesh.animations; // 在每一帧中更新混合器 function update () { mixer.update( deltaSeconds ); } // 播放一个特定的动画 var clip = THREE.AnimationClip.findByName( clips, 'dance' ); var action = mixer.clipAction( clip ); action.play(); // 播放所有的动画 clips.forEach( function ( clip ) { mixer.clipAction( clip ).play(); } );
